## Installation 
### Setting UP

```
$ python3 -m venv venv

$ source venv/bin/activate

$ pip3 install -r requirements.txt

$ python3 manage.py makemigrations

$ python3 manage.py migrate

$ python3 manage.py collectstatic
```
### Run on localhost port 8000
```
python3 manage.py runserver
```
